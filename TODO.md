# 機能改善タスクリスト

## 優先度1（重要・早急に対応すべき）

- [x] **1. 作業日報の編集・詳細画面**
  - 現状: 作業日報には新規作成のみで、編集・詳細表示ページがない
  - 対応: `/work-report/[id]` (詳細) と編集機能を追加

- [x] **2. 承認ワークフローUI**
  - 現状: 承認テーブルはDBに存在するが、UIがない
  - 対応: 管理者が日報を承認/差戻しできる画面、ステータス表示

- [x] **3. 日報の削除機能**
  - 現状: 営業日報・作業日報ともに削除APIやUIがない
  - 対応: 確認ダイアログ付きの削除ボタン（本人または管理者のみ）

- [x] **4. PDF出力・印刷機能**
  - 現状: 画面表示のみで紙への出力手段がない
  - 対応: 日報を印刷用レイアウトでPDF出力、一括印刷も対応

## 優先度2（業務効率化）

- [x] **5. 分析ダッシュボード**
  - 月別/週別の日報件数推移グラフ
  - 訪問先ランキング、案件別集計
  - 作業日報の工数・材料費集計

- [x] **6. 高度な検索・フィルタリング**
  - 期間指定、訪問先名、案件名での絞り込み
  - キーワード全文検索

- [x] **7. 案件管理の強化**
  - 案件の編集・アーカイブ機能
  - 案件ごとの進捗率管理
  - 案件に紐づく日報の一覧表示

- [x] **8. 通知機能**
  - 日報未提出リマインダー（管理者向け）
  - 承認完了通知（作成者向け）

## 優先度3（あると便利）

- [ ] **9. 日報テンプレート機能**
  - よく使うパターンをテンプレートとして保存
  - テンプレートから素早く日報作成

- [x] **10. 自動下書き保存**
  - 入力中のデータを定期的にlocalStorageに保存
  - ブラウザを閉じても復元可能

- [ ] **11. ファイル添付機能**
  - 写真や資料を日報に添付
  - 作業現場の写真記録に有用

- [ ] **12. メール送信機能**
  - 日報をメールで上長に自動送信
  - 日次/週次の集計メール
