## 1. はじめに

### 1.1 システム概要

本システムは、安島工業株式会社の日報管理を行うWebアプリケーションです。
スマートフォン・タブレット・パソコンのいずれからでもご利用いただけます。

### 1.2 対応する日報の種類

| 日報種類 | 用途 |
|---------|------|
| **営業日報** | 営業活動（訪問先・面接者・営業内容・経費）の記録 |
| **作業日報** | 建築・施工作業（作業者・材料・外注先）の記録 |

### 1.3 ユーザー権限

| 権限 | できること |
|------|-----------|
| **一般ユーザー** | 日報の作成・閲覧・テンプレート管理・個人設定 |
| **管理者** | 上記に加え、承認管理・ユーザー管理・マスタ管理・レポート分析 |

## 2. ログイン・ログアウト

### 2.1 ログイン方法

1. ブラウザでシステムのURLにアクセスします
2. **ユーザー名**（メールアドレス）を入力します
3. **パスワード**を入力します
4. **「ログイン」** ボタンをタップします

> **パスワードの確認**: 入力欄右側の目のアイコンをタップすると、入力中のパスワードを表示できます。

### 2.2 ログアウト方法

画面右上の **赤いログアウトアイコン** をタップします。

## 3. ダッシュボード（TOP画面）

ログイン後に表示されるメイン画面です。

### 3.1 画面構成

#### 上部: 報告書タイプの切り替え
- **「営業日報」タブ** / **「作業日報」タブ** をタップして切り替えます

#### 中央: クイックアクション

| ボタン | 説明 |
|-------|------|
| **新規日報作成** | 新しい日報を作成します |
| **日報一覧** | 過去の日報を確認・編集します |

#### 右側: 統計情報
- 今月の作成数・承認待ち数・承認済み数が表示されます

#### 下部: クイックリンク

| リンク | 説明 |
|-------|------|
| **通知** | 未読通知の確認（未読数がバッジ表示されます） |
| **テンプレート管理** | 作業日報のテンプレートを管理します |
| **一括日報作成** | テンプレートで複数日分の日報を一括作成します |
| **レポート・分析** | 月別の集計レポートを確認します |
| **アカウント設定** | ユーザー名・パスワードを変更します |

### 3.2 未提出リマインダー（管理者のみ）

管理者には、当日の日報が未提出のユーザー名がオレンジ色のバナーで表示されます。

## 4. 営業日報

### 4.1 新規作成

1. ダッシュボードで **「営業日報」タブ** を選択します
2. **「新規日報作成」** をタップします
3. 以下の項目を入力します

#### 基本情報
| 項目 | 入力方法 |
|------|---------|
| **日付** | 日付を選択（初期値は当日） |
| **氏名** | ログインユーザー名が自動入力されます |

#### 訪問記録（最大16件）
| 項目 | 入力方法 | 必須 |
|------|---------|------|
| **訪問先** | ドロップダウンから選択、または直接入力 | ○ |
| **面接者ご氏名** | テキスト入力 | |
| **作業開始時刻** | 30分刻みで選択 | |
| **作業終了時刻** | 30分刻みで選択（開始の1時間後が自動設定） | |
| **営業内容** | テキスト入力（500文字まで） | ○ |
| **支出経費（円）** | 数字を入力 | |

- **訪問記録を追加**: 画面下部の「+ 訪問記録を追加」ボタンで追加できます
- **訪問記録を削除**: 各記録右上の「削除」ボタンで削除できます

#### 特記事項
連絡事項・備考を入力できます（500文字まで）。

4. **「保存」** ボタンをタップして送信します
5. 送信成功後、自動的にダッシュボードに戻ります

### 4.2 一覧の確認

1. ダッシュボードで **「日報一覧」** をタップします
2. 過去の営業日報が日付順に表示されます

#### ステータスの見方
| 表示 | 意味 |
|------|------|
| **承認済み**（緑） | すべての承認者が承認済み |
| **承認待ち**（オレンジ） | 承認者の承認を待っている状態 |
| **一部承認**（黄） | 一部の承認者が承認済み |
| **差戻し**（赤） | 承認者により差し戻された状態 |

### 4.3 編集・削除

- 一覧から日報をタップすると詳細画面に遷移します
- 下書き状態の日報は編集・削除が可能です
- 承認済みの日報は閲覧のみとなります

### 4.4 印刷

1. 日報の詳細画面を開きます
2. **「印刷」** ボタンをタップします
3. 印刷プレビュー画面が表示されます（A4サイズ・承認欄付き）
4. **「印刷する」** ボタンでブラウザの印刷機能が起動します

## 5. 作業日報

### 5.1 新規作成

1. ダッシュボードで **「作業日報」タブ** を選択します
2. **「新規日報作成」** をタップします
3. 物件一覧画面が表示されます
4. 日報を作成する **物件名をタップ** します
5. 日報作成画面に遷移します

#### 基本情報
| 項目 | 入力方法 | 必須 |
|------|---------|------|
| **日付** | 日付を選択 | ○ |
| **工事名** | 選択した物件名が自動入力されます | ○ |
| **工事種別** | ドロップダウンから選択 | |
| **工事番号** | テキスト入力 | |
| **天候** | ドロップダウンから選択（晴天・曇天・雨 等） | |
| **氏名** | ログインユーザー名が自動入力されます | |

#### 作業者記録（最大11件）
| 項目 | 入力方法 |
|------|---------|
| **氏名** | テキスト入力（必須） |
| **開始時刻** | HH:MM形式で入力 |
| **終了時刻** | HH:MM形式で入力 |
| **工数（人工）** | 数値を入力 |
| **工種** | テキスト入力またはドロップダウン選択 |
| **内訳** | テキスト入力 |
| **当日工数** | 数値を入力 |
| **累計工数** | 数値を入力 |

> **前日コピー**: 「前日コピー」ボタンをタップすると、前日の作業者記録が一括でコピーされます。同じメンバーで作業する場合に便利です。

#### 使用材料・消耗品（最大5件）
| 項目 | 入力方法 |
|------|---------|
| **材料名** | テキスト入力またはマスタから選択（必須） |
| **容量** | テキスト入力 |
| **容量単位** | ドロップダウン選択（L、mL、m、kg 等） |
| **数量** | 数値を入力 |
| **単価** | 数値を入力 |
| **金額** | 数量 x 単価で自動計算されます |
| **外注先** | テキスト入力またはマスタから選択 |

> 材料の小計（総金額）は画面下部に自動表示されます。

#### 外注先（最大10件）
| 項目 | 入力方法 |
|------|---------|
| **外注先名** | テキスト入力またはマスタから選択 |
| **人数** | 数値を入力 |
| **作業内容** | テキスト入力 |

#### 遠隔地・交通誘導警備員（任意）
| 項目 | 入力方法 |
|------|---------|
| **遠隔地出発時刻** | HH:MM形式 |
| **現場着時刻** | HH:MM形式 |
| **現場発時刻** | HH:MM形式 |
| **会社着時刻** | HH:MM形式 |
| **交通誘導警備員人数** | 数値 |
| **警備員開始時刻** | HH:MM形式 |
| **警備員終了時刻** | HH:MM形式 |

#### 連絡事項
連絡事項・備考を入力できます（500文字まで）。

6. **「保存」** ボタンをタップして送信します

### 5.2 下書き自動保存

作業日報の入力中は、内容が自動的に下書き保存されます。
入力途中でブラウザを閉じても、次回アクセス時に復元されます。

### 5.3 物件一覧のフィルター

物件一覧画面では、以下のフィルターが使用できます。

| フィルター | 説明 |
|-----------|------|
| **進行中** | 現在進行中の物件のみ表示 |
| **完了** | 完了した物件のみ表示 |
| **アーカイブ** | アーカイブ済みの物件のみ表示 |
| **検索** | 物件名で絞り込み |

### 5.4 印刷

営業日報と同様に、詳細画面から印刷プレビューを開いて印刷できます。

## 6. テンプレート機能

よく使う作業日報の入力パターンをテンプレートとして保存できます。

### 6.1 テンプレートの保存

1. 作業日報の作成画面で内容を入力します
2. **「テンプレートとして保存」** を選択します
3. テンプレート名を入力して保存します

### 6.2 テンプレートの利用

1. ダッシュボードの **「テンプレート管理」** をタップします
2. 使用したいテンプレートの **「複製」** をタップします
3. テンプレートの内容が反映された日報作成画面に遷移します
4. 必要に応じて修正し、保存します

### 6.3 テンプレートの管理

テンプレート一覧画面から以下の操作が可能です。

| 操作 | 説明 |
|------|------|
| **複製** | テンプレートから新規日報を作成 |
| **編集** | テンプレートの内容を修正 |
| **共有** | 全従業員がこのテンプレートを利用可能にする |
| **削除** | テンプレートを削除（確認ダイアログが表示されます） |

---

## 7. 一括日報作成

テンプレートを使って、複数の日付の作業日報をまとめて作成できます。

### 7.1 操作手順

1. ダッシュボードの **「一括日報作成」** をタップします
2. 使用する **テンプレート** を選択します
3. 作成対象の **月** を選択します
4. カレンダーから **対象日付をチェック** します（複数選択可能）
   - 既に日報がある日はグレーアウトされ選択できません
5. **「一括作成」** ボタンをタップします
6. 作成結果が表示されます（例：「5件作成、2件スキップ」）

## 8. 通知機能

### 8.1 通知の確認

1. ダッシュボードの **ベルアイコン** または **「通知」リンク** をタップします
2. 通知一覧が表示されます

### 8.2 通知の種類

| 通知タイプ | 内容 |
|-----------|------|
| **日報提出** | 日報が提出された時（承認者向け） |
| **承認完了** | 日報が承認された時（作成者向け） |
| **差戻し** | 日報が差し戻された時（作成者向け） |
| **リマインダー** | 日報の未提出リマインダー |

### 8.3 通知の操作

| 操作 | 説明 |
|------|------|
| **タップ** | 通知を既読にし、対象の日報に遷移します |
| **全件既読** | すべての通知を一括で既読にします |
| **フィルター** | 未読・既読・すべてで絞り込みできます |

## 9. レポート・分析

### 9.1 アクセス方法

ダッシュボードの **「レポート・分析」** をタップします。

| 項目 | 内容 |
|------|------|
| **材料種類数** | 使用された材料の種類数 |
| **総金額** | 材料費の合計金額 |
| **材料別集計** | 材料ごとの数量・金額・使用現場数 |

#### 外注先集計レポート
| 項目 | 内容 |
|------|------|
| **外注先数** | 利用した外注先の数 |
| **総人数** | 外注先の延べ人数 |
| **外注先別集計** | 外注先ごとの人数・日数・作業内容 |

### 9.3 月の切り替え

画面上部の **◀ ▶** ボタンで前月・翌月に切り替えられます。

## 10. アカウント設定

### 10.1 アクセス方法

- ダッシュボードの **「アカウント設定」** をタップ
- または画面右上の **歯車アイコン** をタップ

### 10.2 ユーザー名の変更

1. 「ユーザー名変更」セクションで新しいメールアドレスを入力します
2. **「ユーザー名を更新」** をタップします

### 10.3 パスワードの変更

1. **現在のパスワード** を入力します
2. **新しいパスワード** を入力します（8文字以上）
3. **確認用パスワード** を再度入力します
4. **「パスワードを変更」** をタップします

> 新しいパスワードと確認用パスワードが一致しない場合、エラーが表示されます。

### 11.3 承認管理

#### 承認画面の操作
1. **「承認管理」** をタップします
2. 承認待ちの日報が一覧表示されます

#### フィルター機能
| フィルター | 説明 |
|-----------|------|
| **ユーザー** | 特定ユーザーの日報のみ表示 |
| **ステータス** | 承認待ち・承認済み・差戻しで絞り込み |
| **月選択** | 月ごとに表示 |
| **検索** | 報告内容のキーワード検索 |

#### 承認の実行
1. 日報の **「詳細」** をタップして内容を確認します
2. **「承認」** ボタンをタップして承認します
3. 差し戻す場合は **「差戻し」** ボタンをタップし、コメントを入力します

#### 承認ルートの表示
各日報には承認ルート（例：社長 → 常務 → 部長）が表示され、各承認者の承認状況が確認できます。

### 11.4 ユーザー管理

#### 新規ユーザーの作成
1. **「新規作成」** ボタンをタップします
2. 以下の情報を入力します

| 項目 | 入力方法 | 必須 |
|------|---------|------|
| **氏名** | テキスト入力 | ○ |
| **ユーザー名** | メールアドレス形式 | ○ |
| **パスワード** | 8文字以上 | ○ |
| **役職** | ドロップダウン選択（部長・課長・係長 等） | |
| **権限** | 一般ユーザー / 管理者 | |
| **デフォルト日報** | 営業日報 / 作業日報 / 両方 | |

3. **「作成する」** をタップします

#### ユーザーの編集
- ユーザー一覧で対象ユーザーの **編集アイコン** をタップします
- 役職・権限・デフォルト日報を変更できます

#### ユーザーの無効化・有効化
- **無効化**: アカウントを一時的に利用停止にします（ログイン不可）
- **有効化**: 無効化したアカウントを再度利用可能にします

#### ユーザーの削除
- 日報が紐づいていないユーザーのみ削除可能です
- 日報が存在するユーザーは「無効化」をご利用ください

### 11.5 案件管理

#### 新規案件の登録
| 項目 | 入力方法 |
|------|---------|
| **工事名** | テキスト入力（必須） |
| **工事種別** | ドロップダウン選択 |
| **工事番号** | テキスト入力 |
| **発注者** | テキスト入力 |
| **現場住所** | テキスト入力 |
| **工期** | 開始日・終了日を選択 |
| **進捗率** | 0〜100%で設定 |

#### ステータス管理
| ステータス | 説明 |
|-----------|------|
| **進行中** | 現在施工中の案件 |
| **完了** | 施工完了した案件 |
| **アーカイブ** | 管理上保管している案件 |

### 11.6 マスタ管理

以下のマスタデータを管理画面から登録・編集できます。

| マスタ | 管理内容 |
|-------|---------|
| **工事種別** | 建築塗装工事、鋼橋塗装工事、防水工事 等 |
| **使用材料** | 材料名・デフォルト容量・デフォルト単位 |
| **外注先** | 外注先名 |
| **単位** | L、mL、m、cm、kg、g、本、枚 等 |

各マスタで共通の操作:
- **追加**: 新しい項目を追加します
- **有効/無効**: 切り替えにより日報入力時の選択肢に表示/非表示を制御します
- **削除**: 不要な項目を削除します

### 11.7 承認ルート設定

日報の承認フローを設定します。

#### 承認ルートの作成
1. ルート名を入力します（例：「標準承認ルート」）
2. 承認者の役職を順番に追加します（例：部長 → 常務 → 社長）
3. デフォルトルートに設定するかを選択します

#### 承認の仕組み
- 設定された役職の順番で承認が進みます
- すべての承認者が承認すると「承認済み」になります
- いずれかの承認者が差し戻すと「差戻し」になります

### 11.8 操作ログ

システム上のすべての操作履歴を確認できます。

| 表示項目 | 内容 |
|---------|------|
| **操作者** | 操作を行ったユーザー名 |

## 12. よくある質問（FAQ）

### Q1. ログインできません
**A.** ユーザー名（メールアドレス）とパスワードが正しいか確認してください。
それでもログインできない場合は、管理者にアカウントの状態（有効/無効）を確認してもらってください。

### Q2. パスワードを忘れました
**A.** 管理者に連絡して、パスワードのリセットを依頼してください。

### Q3. 日報を保存したのに表示されません
**A.** 保存時にエラーメッセージが表示されていなかったか確認してください。
ネットワーク接続が不安定な場合、保存が失敗することがあります。再度保存をお試しください。

### Q4. 承認された日報を修正したい
**A.** 承認済みの日報は編集できません。管理者に差戻しを依頼してください。
差し戻された後、編集・再提出が可能になります。

### Q5. 前日と同じ作業者で入力したい
**A.** 作業日報の作成画面で **「前日コピー」** ボタンをタップすると、
前日の作業者記録が自動的にコピーされます。

### Q6. スマートフォンで経費が入力できません
**A.** 支出経費欄は数字のみ入力可能です。
数字キーボードが表示されない場合は、入力欄を長押ししてから入力をお試しください。

### Q7. 印刷がうまくいきません
**A.** 印刷プレビュー画面で「印刷する」ボタンをタップすると、
ブラウザの印刷ダイアログが表示されます。
用紙サイズは **A4** を選択してください。

### Q8. 通知が届きません
**A.** 通知はシステム内で確認できます。ダッシュボードのベルアイコンをご確認ください。
メール通知が必要な場合は、管理者にシステム設定を確認してもらってください。
